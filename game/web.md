# 웹 게임을 만들며 배우는 자바스크립트 
# 내용 정리 🍽

## BOM (Browser Object Model)
- 브라우저의 정보에 접근하거나 브라우저의 여러 기능들을 제어할 수 있는 객체 모델

## Window 객체
- 웹 브라우저의 창을 나타내는 객체
- 자바스크립트의 모든 객체, 전역 함수, 전역 변수들은 자동으로 window 객체의 프로퍼티
 
 ## Document 객체
- window 객체 내 가장 중요한 프로퍼티 중 하나
- 브라우저 창에 표시되는 내용에 해당하는 문서를 나타내는 객체
